## Getting Started
### Prerequisites
* npm
  ```sh
  npm install npm@latest -g
  ```
* pip 
  ```sh
  sudo apt install python3-pip
  ```

### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/clarion22/EasyMeals
   ```
2. Change directory 
   ```
   cd EasyMeals
   ```
3. Create an env file 
   ```
   touch .env
   ```
4. Change directory and Install Dependencies
   ```
   cd react-app && npm install 
   ```
